== Description ==

IronSky is provided by IronPlatform. 

This plugin shows a 3d mapbox’s map and loads realtime flights as map’s features.
The map is loaded using Mapbox and flights data is loaded from Airlabs.

Plugin's website: https://ironplatform.com.uy/ironplatform/ironsky/
Developer info and portfolio: https://ironplatform.com.uy/ironteam/pablo-pignolo/
Check IronPlatform's free apps here: https://ironplatform.com.uy/

To use it, you need to set both apikeys to use this plugin. 
Free ApiKeys are supported. Get them from:

- mapbox https://account.mapbox.com/
- airlabs.co https://airlabs.co/signup

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/ironsky directory, or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the 'IronSky' settings screen to configure the plugin.

== Frequently Asked Questions ==

= How do I install IronSky? =

To install IronSky, you can either upload the plugin files to the /wp-content/plugins/ironsky directory
via FTP or install the plugin directly through the WordPress plugins screen. 
After uploading, go to the 'Plugins' screen in WordPress to activate it.

= How do I configure the IronSky plugin? =

Once the plugin is activated, go to the 'IronSky' settings screen in your WordPress admin panel. 
Here, you can configure various settings like API keys, map dimensions, and initial map coordinates.

= Where can I find my API keys for Aviationstack and Mapbox? =

You can obtain your Airlabs API key by signing up at their website. https://airlabs.co/signup
For Mapbox, create an account on the Mapbox website to receive your API key. https://account.mapbox.com/

= How often are the flight data on the map updated? =

The flight data is refreshed periodically based on the settings you configure in the IronSky plugin. 
You can set the refresh rate according to your preferences and API limitations.

= Can I customize the appearance of the map in IronSky? =

Yes, IronSky allows customization of map dimensions, zoom levels, and initial positioning. 
Further styling can be done through additional CSS and Mapbox settings.

= What should I do if I encounter errors or issues with the plugin? =

First, ensure that all settings, especially API keys, are correctly entered. 
If problems persist, you can seek support on the plugin's forum page or contact the developer directly.
Check https://ironplatform.com.uy/ironplatform/ironsky/

= IronSky compatible with all WordPress themes? =

IronSky is designed to be compatible with most WordPress themes. 
However, if you encounter any display issues, it might be due to conflicts with your specific theme 
or other plugins.
